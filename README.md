# NASA ASRS Downloader

Tool to programmatically download report data from the [NASA ASRS](https://asrs.arc.nasa.gov/).

## Quick Start

1.  **Install dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run**:
    ```bash
    python3 asrs_downloader.py --start-year 2024 --start-month 1 --end-year 2024 --end-month 12
    ```
